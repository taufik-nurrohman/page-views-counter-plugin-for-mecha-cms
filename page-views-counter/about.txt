Title: Page Views Counter
Author: Taufik Nurrohman
URL: http://latitudu.com
Version: 1.2.0

====

> This plugin will count the number of your page visits.

Your page visits will not be counted if you&rsquo;re logged in.

The _Ranges_ field is used for adding leading zeros as the value specified in the field. If the previous statistic shows the number of page views as **35 Page Views**, then after you put `5` on the field, the statistic will becomes **00035 Page Views**.

#### Widget Installation

Put this widget code in your `article.php`, `page.php` or `index.php` file:

~~~ .php
<?php echo Widget::pageViews('page-slug', 'Views'); ?>
~~~

##### Example for `article.php`

~~~ .html
<footer class="post-footer">
  <?php echo Widget::pageViews($article->slug, 'Views'); ?>
</footer>
~~~

##### Example for `page.php`

~~~ .html
<footer class="post-footer">
  <?php echo Widget::pageViews($page->slug, 'Views'); ?>
</footer>
~~~

##### Example for `index.php`

~~~ .html
<?php foreach($articles as $article): ?>

…

<footer class="post-footer">
  <?php echo Widget::pageViews($article->slug, 'Views'); ?>
</footer>

…

<?php endforeach; ?>
~~~

#### Advanced

Now you can display the page views statistic data that are stored in a folder other than `cargo\articles` and `cargo\pages` by entering the name of the folders in the `$slug` parameter:

~~~ .php
echo Widget::pageViews('2015/02/10/page-slug', 'Views');
~~~

This plugin will automatically stores the page views data of your custom pages (if any). For example, if you have a new public page with address ` {{url}}2015/02/10/page-slug `, then this plugin will automatically creates new folders in the `cargo` folder for storing the page views statistics by using the guidelines of the page URL:

~~~ .no-highlight
cargo
├── __2015/
│   └── __02/
│       └── __10/
│           └── __page-slug.txt
├── articles/
└── pages/
~~~
