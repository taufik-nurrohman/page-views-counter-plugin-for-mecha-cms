Title: Penghitung Tayangan Halaman
Author: Taufik Nurrohman
URL: http://latitudu.com
Version: 1.2.0

====

> Plugin ini akan menghitung jumlah penayangan halaman Anda.

Penayangan halaman Anda tidak akan dihitung jika Anda sedang masuk (_logged in_).

Bidang _Rentang_ digunakan untuk menambahkan awalan berupa angka nol sebanyak nilai yang dicantumkan di dalam bidang tersebut. Jika sebelumnya statistik penayangan halaman menunjukkan angka **35 Penayangan**, maka setelah Anda memasukkan angka `5` ke dalam bidang tersebut, statistik penayangan halaman akan berubah menjadi **00035 Penayangan**.

#### Pemasangan Widget

Letakkan kode widget ini pada berkas `article.php`, `page.php` atau `index.php`:

~~~ .php
<?php echo Widget::pageViews('slug-halaman', 'Penayangan'); ?>
~~~

##### Contoh untuk `article.php`

~~~ .html
<footer class="post-footer">
  <?php echo Widget::pageViews($article->slug, 'Penayangan'); ?>
</footer>
~~~

##### Contoh untuk `page.php`

~~~ .html
<footer class="post-footer">
  <?php echo Widget::pageViews($page->slug, 'Penayangan'); ?>
</footer>
~~~

##### Contoh untuk `index.php`

~~~ .html
<?php foreach($articles as $article): ?>

…

<footer class="post-footer">
  <?php echo Widget::pageViews($article->slug, 'Penayangan'); ?>
</footer>

…

<?php endforeach; ?>
~~~

#### Tingkat Lanjut

Sekarang Anda bisa menampilkan data statistik tayangan halaman Anda yang tersimpan di dalam folder selain `cargo\articles` dan `cargo\pages` dengan cara memasukkan nama-nama folder tersebut pada parameter `$slug`:

~~~ .php
echo Widget::pageViews('2015/02/10/slug-halaman', 'Penayangan');
~~~

Plugin ini akan secara otomatis menyimpan data-data statistik tayangan halaman kustom Anda (jika ada). Sebagai contoh, jika Anda memiliki halaman publik baru dengan alamat menuju ` {{url}}2015/02/10/slug-halaman `, maka plugin ini dapat secara otomatis membuatkan folder-folder baru di dalam folder `cargo` untuk menyimpan data-data statistik halaman tersebut dengan cara menggunakan panduan dari URL halaman tersebut:

~~~ .no-highlight
cargo
├── __2015/
│   └── __02/
│       └── __10/
│           └── __slug-halaman.txt
├── articles/
└── pages/
~~~
